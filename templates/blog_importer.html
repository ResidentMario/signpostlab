The Wikipedia Signpost occasionally reprints stories which first appeared on the Wikimedia Blog. This script assists with the process by providing machine-assisted translations from the latter format to the former one. For more information on this partnership refer to our content guidance.

Note that while this tool uses presets for the Wikimedia Blog, the HTML-to-Wikitext conversion that it uses works (up to a certain point) on any arbitrary HTML. To use it on some other blog or post you can copy-paste the source into the HTML transform directly.
